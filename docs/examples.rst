.. _examples:

Examples
~~~~~~~~

The following are examples of using ``olwidget.js`` in HTML pages.  See :doc:`olwidget.js` for standalone javascript documentation, and :doc:`django-olwidget` for use in Django.

In addition, the source distribution contains an example project called ``test_project`` which illustrates use of the Django app.

Editable Maps
-------------
* `Simple example <_static/examples/simple.html>`_
* `Collections <_static/examples/collection.html>`_ of multiple geometries (e.g. more
  than one linestring at once).
* Read only maps and `multiple maps per page <_static/examples/read_only_and_multiple_maps.html>`_
* Custom `starting locations and colors <_static/examples/custom_start_point_and_colors.html>`_
* `Other map providers <_static/examples/other_providers.html>`_, including Google,
  Yahoo, Bing Maps (formerly Microsoft VE), OpenStreetMaps, MetaCarta, and
  CloudMade.
* `Multiple geometry types <_static/examples/multiple_geometries.html>`_
* `Other projections <_static/examples/other_projections.html>`_

Info Maps
